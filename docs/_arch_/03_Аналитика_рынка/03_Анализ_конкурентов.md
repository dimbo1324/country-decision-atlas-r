РАЗДЕЛ 4. ГЛАВНЫЙ ВЫВОД О КОНКУРЕНТНОМ ПОЛЕ
════════════════════════════════════════════════════════════════════

  НИ ОДНА ПЛАТФОРМА не объединяет одновременно:
    1. Объективные страновые индикаторы
    2. Визовые/резидентные/гражданские пути
    3. Отслеживание правовых и политических изменений
    4. Структурированный человеческий опыт переезда
    5. Персонализированное сценарное сопоставление
    6. Объяснимое скоринговое ранжирование
    7. Тренд страны (Country Drift) во времени
    8. Ссылки на источники с доказательствами
    9. Алерты и watchlists

  Существующие сильные решения фрагментированы по сегментам.
  Рыночная ниша очевидна: гибридная платформа, которая рассматривает
  выбор страны как decision-support workflow, а не как статическую
  статью, форум, таблицу стоимости жизни или воронку визового сервиса.

  Numbeo API Basic         ~$260/мес.

  LegiScan (national API)  ~$6,000/год

  Remote (EOR)             $599/сотрудника/мес.
  Deel (contractor)        $49/контракт/мес.

  Localyze, Jobbatical     По запросу (custom pricing)

  КОНСАЛТИНГ:
  Henley & Partners        Большой капитал + advisory fees
  Global Citizen Solutions По запросу
  Immigration lawyers      $150–$500+/час (зависит от юрисдикции)

  ВЫВОД ДЛЯ CDA:
  • B2C freemium + premium подписка: $9–49/мес. или $49–199/год

  • B2B API: $300–1,000+/мес. по образцу Numbeo
  • Маркетплейс: 10–20% комиссии с транзакций
