## Краткая формула проекта

Country Decision Atlas — это доказательная платформа выбора страны для жизни, миграции, работы, бизнеса, инвестиций и долгосрочного статуса.

Главная формула продукта:

**объективные данные + правовые сигналы + источники + реальные истории + сценарное сравнение + объяснимый decision engine + мультиязычный интерфейс.**
