# Архив проекта Country Decision Atlas (`docs/_arch_`)

Это временное хранилище справочных материалов проекта: видение продукта, архитектура, методология, рыночные и миграционные исследования, конкурентный анализ и аудит. Раздел приведён к единому виду — на русском языке, без дубликатов, с логичной структурой. Это не рабочая документация репозитория и не влияет на код; материалы служат обоснованием продуктовых и архитектурных решений.

> Раздел планируется удалить в будущем. Пока он сохранён как единый упорядоченный справочник для разработчиков и ИИ-помощников.

## Структура

| Путь | Содержание |
|---|---|
| `00_Эпизод_0_рабочий_стандарт.md` | Глобальный рабочий стандарт проекта: ветки, quality gate, порядок слоёв, запреты, архитектурные инварианты, формат отчёта |
| `01_Описание_проекта/` | Полное продуктово-архитектурное описание Country Decision Atlas (v0.4): видение, аудитория, сценарии, MVP, модули, бэкенд, данные, фронтенд, мультиязычность, дорожная карта, критерии приёмки |
| `02_Архитектура_и_методология/` | Архитектура и визуальная система; методология индекса Country Intelligence Index (научные основы, уникальные метрики) |
| `03_Аналитика_рынка/` | Аналитический пакет фазы исследований: обзор рынка, боли пользователей и спрос, конкуренты, монетизация, продуктовые возможности |
| `04_Исследование_миграции/` | Сводное исследование из 120 вопросов о международной миграции и выборе страны (динамика, мотивы, поиск, сообщества, критерии, законы, сервисы, монетизация, проблемы, тренды) и индекс из 50 источников |
| `05_Тренды_рынка_мобильности/` | Рыночно-стратегический обзор глобальной мобильности: оценки рынка, демография, правовой курс, позиционирование продукта |
| `06_Глубокие_исследования/` | Продуктовые выводы и модель данных: объектная модель, индекс направления страны, модель сигналов, решение о готовности MVP, продуктовые модули |
| `07_Конкурентный_анализ/` | Анализ Quora, Trustpilot, Glassdoor и заимствуемые продуктовые паттерны для миграционной платформы |
| `08_Аудит_проекта.md` | Подробный аудит проекта |
| `09_План_реализации_проекта.md` | Исполнительный план разработки от Эпизода 4 до полного MVP и горизонта (источник истины по порядку эпизодов) |

## Формула проекта

Country Decision Atlas — доказательная платформа выбора страны для жизни, миграции, работы, бизнеса, инвестиций и долгосрочного статуса:

**объективные данные + правовые сигналы + источники + реальные истории + сценарное сравнение + объяснимый decision engine + мультиязычный интерфейс.**
