# Country Priority Matrix

## Selection logic

The first countries should be selected for maximum learning value. A good first set must include different types of migration decisions:

- skilled-worker immigration;
- digital nomad / remote worker routes;
- founder / business / investor routes;
- high-demand lifestyle countries;
- countries with recent policy changes;
- countries with strong official data;
- countries with many user stories;
- countries with high uncertainty.

## Country page template

Each country should have:

1. Overview.
2. Best-fit personas.
3. Main visa/residence routes.
4. Route comparison.
5. Cost baseline.
6. Housing reality.
7. Work/business environment.
8. Tax headline.
9. Family/healthcare/school.
10. Safety and governance.
11. Citizenship/PR path.
12. Recent law changes.
13. Country direction timeline.
14. Real case examples.
15. Sources and confidence.
16. Editor notes.
17. Last checked date.

## Data confidence by country

### Likely easier
- Germany;
- Canada;
- UK;
- Netherlands;
- Japan;
- Singapore.

Reason: official data, English documentation, structured government sources.

### Medium difficulty
- Spain;
- Portugal;
- UAE;
- Mexico;
- South Korea;
- Uruguay.

Reason: multiple languages, route complexity, regional/local variation.

### Harder
- Thailand;
- Saudi Arabia;
- some second-wave countries.

Reason: route interpretation, local implementation, language, rapid changes, consultant-driven information.
