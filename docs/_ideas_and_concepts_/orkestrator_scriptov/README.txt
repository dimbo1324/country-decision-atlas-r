Python Script Orchestrator Demo
===============================

How to run:

1. Open a terminal.
2. Go to this folder:

   cd root

3. Run the main orchestrator:

   python main_script.py

4. Choose a script:

   1 or script1 - prints Hello World
   2 or script2 - generates two random numbers and prints their sum
   3 or script3 - prints a small system/runtime demo

If you press Enter without typing anything, script1 runs automatically.

No external dependencies are required.
